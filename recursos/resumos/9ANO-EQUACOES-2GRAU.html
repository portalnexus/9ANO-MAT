Equações do 2º Grau

1. Conceitos Fundamentais

Definição 1.1 (Equação do 2º Grau)

Uma equação do 2º grau na variável @x@ é toda equação que pode ser escrita na forma reduzida:

@@ ax^2 + bx + c = 0 @@

onde @a@, @b@ e @c@ são números reais, com @a \neq 0@.

Observação: A condição @a \neq 0@ é essencial. Se @a = 0@, a equação se reduz a @bx + c = 0@, que é uma equação do 1º grau.

Exemplos:

  • @x^2 - 5x + 6 = 0@ é uma equação do 2º grau com @a = 1@, @b = -5@, @c = 6@
  • @2x^2 + 3x - 1 = 0@ é uma equação do 2º grau com @a = 2@, @b = 3@, @c = -1@
  • @-x^2 + 4 = 0@ é uma equação do 2º grau com @a = -1@, @b = 0@, @c = 4@
  • @3x^2 - 7x = 0@ é uma equação do 2º grau com @a = 3@, @b = -7@, @c = 0@

Definição 1.2 (Coeficientes da Equação)

Na equação @ax^2 + bx + c = 0@, os números reais @a@, @b@ e @c@ são chamados de coeficientes:

  • (i) @a@ é o coeficiente do termo quadrático ou coeficiente de @x^2@
  • (ii) @b@ é o coeficiente do termo linear ou coeficiente de @x@
  • (iii) @c@ é o termo independente ou termo constante

Definição 1.3 (Equações Completas e Incompletas)

Uma equação do 2º grau @ax^2 + bx + c = 0@ é chamada de:

  • (i) Completa, se @b \neq 0@ e @c \neq 0@ (todos os coeficientes são não nulos)
  • (ii) Incompleta, se @b = 0@ ou @c = 0@ (ou ambos)

Exemplos de equações incompletas:

  • @x^2 - 16 = 0@ (incompleta com @b = 0@)
  • @2x^2 + 5x = 0@ (incompleta com @c = 0@)
  • @3x^2 = 0@ (incompleta com @b = 0@ e @c = 0@)

Definição 1.4 (Raízes ou Soluções)

As raízes (ou soluções) de uma equação do 2º grau @ax^2 + bx + c = 0@ são os valores de @x \in \mathbb{R}@ que tornam a sentença verdadeira, isto é, que satisfazem a igualdade.

O conjunto de todas as raízes é chamado de conjunto solução e denotado por @S@.

Exemplo: Para a equação @x^2 - 5x + 6 = 0@, os valores @x = 2@ e @x = 3@ são raízes, pois:

  • Para @x = 2@: @2^2 - 5(2) + 6 = 4 - 10 + 6 = 0@ ✓
  • Para @x = 3@: @3^2 - 5(3) + 6 = 9 - 15 + 6 = 0@ ✓

Portanto, @S = \{2, 3\}@.

2. Métodos de Resolução

2.1. Equações Incompletas

As equações incompletas podem ser resolvidas por métodos mais simples do que a fórmula geral.

Caso 1: @b = 0@ (equação da forma @ax^2 + c = 0@)

Método: Isolar @x^2@ e extrair a raiz quadrada.

@@ ax^2 + c = 0 \Rightarrow x^2 = -\frac{c}{a} @@
  • (i) Se @-\frac{c}{a} > 0@, a equação possui duas raízes reais simétricas: @x = \pm\sqrt{-\frac{c}{a}}@
  • (ii) Se @-\frac{c}{a} = 0@, a equação possui uma raiz real: @x = 0@
  • (iii) Se @-\frac{c}{a} < 0@, a equação não possui raízes reais

Caso 2: @c = 0@ (equação da forma @ax^2 + bx = 0@)

Método: Colocar @x@ em evidência (fatoração).

@@ ax^2 + bx = 0 \Rightarrow x(ax + b) = 0 @@

Pela propriedade do produto nulo, temos:

@@ x = 0 \quad \text{ou} \quad ax + b = 0 \Rightarrow x = -\frac{b}{a} @@

Portanto, as duas raízes são sempre reais: @x_1 = 0@ e @x_2 = -\frac{b}{a}@.

Caso 3: @b = 0@ e @c = 0@ (equação da forma @ax^2 = 0@)

Solução: A única raiz é @x = 0@ (raiz dupla).

@@ ax^2 = 0 \Rightarrow x^2 = 0 \Rightarrow x = 0 @@

2.2. Equações Completas - Fórmula de Bhaskara

Teorema 2.1 (Fórmula de Bhaskara)

As raízes da equação do 2º grau @ax^2 + bx + c = 0@, com @a \neq 0@, são dadas pela fórmula:

@@ x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} @@

Definição 2.2 (Discriminante)

O número @\Delta = b^2 - 4ac@ é chamado de discriminante da equação @ax^2 + bx + c = 0@.

O discriminante determina a natureza e a quantidade de raízes reais da equação.

Teorema 2.3 (Natureza das Raízes)

Seja @ax^2 + bx + c = 0@ uma equação do 2º grau com @a \neq 0@. Então:

  • (i) Se @\Delta > 0@, a equação possui duas raízes reais e distintas: @@ x_1 = \frac{-b + \sqrt{\Delta}}{2a} \quad \text{e} \quad x_2 = \frac{-b - \sqrt{\Delta}}{2a} @@
  • (ii) Se @\Delta = 0@, a equação possui uma raiz real dupla: @@ x_1 = x_2 = \frac{-b}{2a} @@
  • (iii) Se @\Delta < 0@, a equação não possui raízes reais (o conjunto solução é vazio: @S = \emptyset@)
Observação: A fórmula de Bhaskara funciona para qualquer equação do 2º grau, completa ou incompleta. No entanto, para equações incompletas, os métodos da Seção 2.1 são geralmente mais rápidos.

3. Exemplos Resolvidos

Exemplo 1 - Identificação de Coeficientes

Enunciado: Identifique os coeficientes @a@, @b@ e @c@ das seguintes equações e classifique-as em completas ou incompletas:

(a) @3x^2 - 7x + 2 = 0@

(b) @x^2 - 9 = 0@

(c) @-2x^2 + 5x = 0@

Solução:

(a) @3x^2 - 7x + 2 = 0@

Coeficientes: @a = 3@, @b = -7@, @c = 2@

Como @b \neq 0@ e @c \neq 0@, a equação é completa.

(b) @x^2 - 9 = 0@

Reescrevendo: @1 \cdot x^2 + 0 \cdot x - 9 = 0@

Coeficientes: @a = 1@, @b = 0@, @c = -9@

Como @b = 0@, a equação é incompleta.

(c) @-2x^2 + 5x = 0@

Reescrevendo: @-2x^2 + 5x + 0 = 0@

Coeficientes: @a = -2@, @b = 5@, @c = 0@

Como @c = 0@, a equação é incompleta.

Exemplo 2 - Resolução de Equação Incompleta (tipo @ax^2 + c = 0@)

Enunciado: Resolva a equação @x^2 - 16 = 0@.

Solução:

Passo 1: Isolar @x^2@:

@@ x^2 - 16 = 0 \Rightarrow x^2 = 16 @@

Passo 2: Extrair a raiz quadrada de ambos os lados:

@@ x = \pm\sqrt{16} = \pm 4 @@

Resposta: @S = \{-4, 4\}@

Exemplo 3 - Resolução de Equação Incompleta (tipo @ax^2 + bx = 0@)

Enunciado: Resolva a equação @3x^2 - 12x = 0@.

Solução:

Passo 1: Colocar @x@ em evidência:

@@ 3x^2 - 12x = 0 \Rightarrow x(3x - 12) = 0 @@

Passo 2: Aplicar a propriedade do produto nulo:

@@ x = 0 \quad \text{ou} \quad 3x - 12 = 0 @@

Passo 3: Resolver @3x - 12 = 0@:

@@ 3x = 12 \Rightarrow x = 4 @@

Resposta: @S = \{0, 4\}@

Exemplo 4 - Equação Completa com Duas Raízes Distintas (@\Delta > 0@)

Enunciado: Resolva a equação @x^2 - 5x + 6 = 0@.

Solução:

Passo 1: Identificar os coeficientes:

@a = 1@, @b = -5@, @c = 6@

Passo 2: Calcular o discriminante:

@@ \Delta = b^2 - 4ac = (-5)^2 - 4(1)(6) = 25 - 24 = 1 @@

Como @\Delta > 0@, a equação possui duas raízes reais distintas.

Passo 3: Aplicar a fórmula de Bhaskara:

@@ x = \frac{-b \pm \sqrt{\Delta}}{2a} = \frac{-(-5) \pm \sqrt{1}}{2(1)} = \frac{5 \pm 1}{2} @@

Passo 4: Calcular as duas raízes:

@@ x_1 = \frac{5 + 1}{2} = \frac{6}{2} = 3 @@ @@ x_2 = \frac{5 - 1}{2} = \frac{4}{2} = 2 @@

Resposta: @S = \{2, 3\}@

Exemplo 5 - Equação Completa com Raiz Dupla (@\Delta = 0@)

Enunciado: Resolva a equação @x^2 - 6x + 9 = 0@.

Solução:

Passo 1: Identificar os coeficientes:

@a = 1@, @b = -6@, @c = 9@

Passo 2: Calcular o discriminante:

@@ \Delta = b^2 - 4ac = (-6)^2 - 4(1)(9) = 36 - 36 = 0 @@

Como @\Delta = 0@, a equação possui uma raiz real dupla.

Passo 3: Aplicar a fórmula de Bhaskara:

@@ x = \frac{-b}{2a} = \frac{-(-6)}{2(1)} = \frac{6}{2} = 3 @@

Resposta: @S = \{3\}@

Observação: Note que @x^2 - 6x + 9 = (x - 3)^2@, que é um produto notável. Por isso, a raiz @x = 3@ aparece duas vezes (raiz dupla).

Exemplo 6 - Problema Contextualizado

Enunciado: A área de um retângulo é 84 m². O comprimento excede a largura em 5 metros. Quais são as dimensões do retângulo?

Solução:

Passo 1: Definir variáveis. Seja @x@ a largura (em metros).

Então o comprimento é @x + 5@ metros.

Passo 2: Montar a equação usando a fórmula da área:

@@ \text{Área} = \text{largura} \times \text{comprimento} @@ @@ 84 = x(x + 5) @@

Passo 3: Desenvolver e colocar na forma reduzida:

@@ 84 = x^2 + 5x \Rightarrow x^2 + 5x - 84 = 0 @@

Passo 4: Identificar coeficientes e calcular @\Delta@:

@a = 1@, @b = 5@, @c = -84@

@@ \Delta = 5^2 - 4(1)(-84) = 25 + 336 = 361 @@

Passo 5: Aplicar a fórmula de Bhaskara:

@@ x = \frac{-5 \pm \sqrt{361}}{2(1)} = \frac{-5 \pm 19}{2} @@

Passo 6: Calcular as raízes:

@@ x_1 = \frac{-5 + 19}{2} = \frac{14}{2} = 7 @@ @@ x_2 = \frac{-5 - 19}{2} = \frac{-24}{2} = -12 @@

Passo 7: Interpretar o resultado no contexto do problema:

Como a largura não pode ser negativa, descartamos @x = -12@.

Portanto, a largura é @x = 7@ m e o comprimento é @x + 5 = 12@ m.

Resposta: As dimensões do retângulo são 7 m de largura e 12 m de comprimento.

Verificação: Área = @7 \times 12 = 84@ m² ✓

    • @x^2 - 36 = 0@

    • @5x^2 + 10x = 0@
    • @x^2 + 4x = 0@
  3. Calcule o discriminante @\Delta@ e determine o número de raízes reais (sem resolver):
    • @x^2 + 4x + 4 = 0@
    • @x^2 - 3x + 5 = 0@
    • @2x^2 - 7x + 3 = 0@
  4. Resolva as equações completas usando a fórmula de Bhaskara:
    • @x^2 + 7x + 10 = 0@
    • @x^2 - 8x + 16 = 0@

  5. Resolva a equação @x^2 - 2x - 15 = 0@.
